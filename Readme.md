# LaceSketch Editor

LaceSketch Editor — это кроссплатформенное приложение (Desktop + Web) для создания и редактирования сколков вологодского кружева. Проект написан на Rust с использованием фреймворка Iced для пользовательского интерфейса и компилируется как в нативные приложения, так и в WebAssembly (WASM).
